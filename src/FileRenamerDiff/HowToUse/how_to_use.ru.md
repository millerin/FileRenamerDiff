# Быстрый

1. Спецификация целевого файла: Вы можете перетаскивать файлы в список файлов или искать <kbd>Поиск файлов</kbd> из определенного каталога.
2. Установить шаблон переименования: Установите параметр  Удалить/Заменить деталь в <kbd> Настройки</kbd>.

3. Подтвердить переименование: Выполните <kbd>просмотр</kbd>, чтобы переименовать.

4. Выполнить переименование: Выполните <kbd>Сохранить переименование</kbd>, чтобы переименовать фактический файл.

# Укажите целевой файл

Вы можете указать целевой файл для переименования двумя способами: "Поиск файла" или "Прямая спецификация".

Чтобы удалить файл из списка целевых файлов, нажмите кнопку в левой части строки. Вы также можете опустошить список файлов, нажав на левую верхнюю кнопку.

## Поиск файлов

Ищите файлы в любой папке и добавляйте их в список целевых файлов.

Существует три способа поиска файлов.

- Выберите с помощью диалогового окна значок папки в левом верхнем углу.
- Перетаскивание папки по пути к папке в левом верхнем углу.
- Непосредственно введите путь к папке в левом верхнем углу и нажмите кнопку <kbd>Выполнить поиск файлов</kbd>.

Вы можете указать условия поиска файлов из <kbd>Настройка</kbd> > <kbd>Выполнить поиск файлов</kbd>.

## Непосредственно укажите

Непосредственно укажите файлы для переименования.

Есть два способа указать напрямую.

- Используйте приведенный выше диалог <kbd>Добавить файлы</kbd> для выбора файла.
- Перетащите файл в список файлов.

# Переименовать настройки

Укажите условия переименования файлов.

Порядок выполнения условий - "Удалить" → "Заменить". Если вы хотите удалить файл после "Заменить", оставьте <kbd>После замещения</kbd> пустым в заголовке <kbd>Заменить Шаблоны</kbd>, что то же самое, что и "Удалить".

## Удалить слова

Укажите символы, которые необходимо удалить.

 <kbd>Добавить</kbd> добавляет новую строку. Вы можете удалить строку, нажав на кнопку с правой стороны. Чтобы изменить порядок следования линий, возьмитесь за левый край линии и перетащите его.

Вы можете выбрать "Common Pattern", нажав на нижнюю правую кнопку.

Если установлен флажок "Регулярное выражение", можно указать символы, которые будут удаляться с помощью регулярного выражения.

### Регулярное выражение

Регулярные выражения (Regex) в этом программном обеспечении используют регулярные выражения "Microsoft .NET". Ниже приведен список типичных. Для более подробного объяснения, пожалуйста, обратитесь к следующим ссылкам.

[Regex Quick Reference - Microsoft Docs ](https://docs.microsoft.com/dotnet/standard/base-types/regular-expression-language-quick-reference#character-escapes)

| Regex              | описание                                                     | шаблон    | Вход                      | Выход    |
| ------------------ | ------------------------------------------------------------ | --------- | ------------------------- | -------- |
| `[`ch_group`]`     | Совпадает с любым одиночным персонажем в *ch_group*.         | [ae]      | gr**a**y, L**A**N**E**    | gry,LN   |
| `[`first`-`last`]` | Диапазон персонажей: Соответствует любому одиночному персонажу в диапазоне от *first* до *last*. | [A-Z]     | **R**ocky4                | ocky4    |
| `.`                | Вайлдкард: Подходит для любого персонажа                     | a.e       | w**ate**r                 | wr       |
| `\w`               | Совпадает с любым словесным персонажем                       | \\w       | **A**.**a** **あ**~**ä**- | . ~-     |
| `\s`               | Подходит для любого персонажа из белого пробела              | \\s       | A B　C                    | ABC      |
| `\d`               | Подходит для любой десятичной цифры                          | \\d       | Rocky**4**                | Rocky    |
| `^`                | Спичка должна начинаться с начала имени файла.               | ^r        | **r**ear rock             | ear rock |
| `$`                | Сопоставление должно происходить в конце имени файла         | $r        | rock rea**r**             | rock rea |
| `*`                | Совпадает с предыдущим элементом ноль или более раз          | o*r       | d**oor**,**or**,o,l**r**  | d,,o,l   |
| `+`                | Совпадает с предыдущим элементом один или несколько раз      | o+r       | d**oor**,**or**,o,lr      | d,,o,lr  |
| `?`                | Совпадает с предыдущим элементом ноль или один раз           | o?r       | do**or**,**or**,o,l**r**  | do,,o,l  |
| `{`n`}`            | Совпадает с предыдущим элементом ровно *n* раз.              | [or]{2}   | d**oo**r,**or**,o,lr      | dr,,o,lr |
| `\`escape          | Распознавание escape-символов типа `. `и `*`, как обычные символы. | \\d\\.\\d | 1\_**2\.3**\_45           | 1__45    |

## Заменить шаблоны

Определяет шаблон до и после замены.

 <kbd>Добавить</kbd> добавит новую строку. Вы можете удалить строку, нажав правую кнопку в строке. Чтобы изменить порядок следования линий, возьмитесь за левый край линии и перетащите его.

Вы можете выбрать "Common Pattern", нажав на правую нижнюю кнопку.

Если установлен флажок <kbd>Регулярное выражение</kbd>, то можно указать заменяющую деталь, используя регулярное выражение.

### Регулярное выражение

Регулярное выражение перед заменой такое же, как и <kbd>Удалить слова</kbd>.

Также существует регулярное выражение, которое может быть использовано после замены. Некоторые из наиболее распространенных выражений перечислены ниже. Для более подробного объяснения, пожалуйста, обратитесь к следующим ссылкам.

[Substitutions In Regular Expressions - Microsoft Docs ](https://docs.microsoft.com/dotnet/standard/base-types/substitutions-in-regular-expressions)

| Regex  | описание                                                     | Текст цели      | После замещения | **Вход**       | Выход              |
| ------ | ------------------------------------------------------------ | --------------- | --------------- | -------------- | ------------------ |
| `$0`   | Включить все совпадающие строки в строку замены              | ABC             | \[\$0\]         | x**ABC**x_AxBC | x**\[ABC\]**x_AxBC |
| `$num` | Включает последнюю подстроку, соответствующую группе захвата, идентифицируемую *num*. | \\d\*\(\\d{3}\) | \$1             | A**0012** 34   | A**012** 34        |

| `\f`  | Преобразовать всю Катакану шириной наполовину в полную ширину. | **ｱﾝﾊﾟﾝ ﾊﾞｲｷﾝ**        | **アンパン バイキン**      |

# Подтверждение переименования

 <kbd>просмотр</kbd> позволяет просматривать файлы до и после их переименования.

Описание каждой колонки в списке файлов следующее.

| Название колонки    | Отображаемое содержимое                                      | Функция                                                      |
| ------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| Удалить кнопку      | Удалить кнопку                                               | Удалить строку. Кнопка в заголовке удаляет все файлы.        |
| Старое имя файла    | Имя файла до переименования. Измененная часть становится розовым фоном. |                                                              |
| Новое имя файла     | Имя файла после переименования. Измененная часть становится зеленым фоном. |                                                              |
| Установите флажок   | При наличии изменений отображается флажок.                   | Кнопка в заголовке делает список файлов только с изменениями. |
| Отметить дубликатом | Если дубликаты есть, то отображается дубликат.               | Кнопка в заголовке делает список файлов только с дубликатами. |
| Каталог             | Каталог, к которому принадлежит файл                         | Нажмите, чтобы отобразить файл в проводнике                  |
| Размер              | Размер файла                                                 |                                                              |
| Дата изменения      | Дата и время изменения файла                                 |                                                              |
| Дата создания       | Дата и время создания файла                                  |                                                              |

# Сохранить переименование

<kbd>Сохранить переименование</kbd> переименует действительный файл. Невозможно выполнить, если в списке файлов есть дубликаты.

Если при переименовании каталог, к которому принадлежит файл, будет перезаписан, то он будет удален из списка файлов.
